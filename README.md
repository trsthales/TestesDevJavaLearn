# ChatGPT Java Example
Este repositório contém um exemplo simples de como integrar o ChatGPT numa aplicação Java. 
O exemplo inclui a configuração necessária, código de exemplo e instruções de uso de Enums, Imutabilidade, Pattern Matching, Records e regex.
## Requisitos
- Java 17
